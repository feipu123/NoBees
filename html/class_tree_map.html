Detailed Description

template<class K, class V>
class TreeMap< K, V >

A map is a sequence of (key, value) entries that provides fast retrieval based on the key. At most one value is held for each key.

TreeMap is the balanced-tree implementation of map. The iterators must iterate through the map in the natural order (operator<) of the key.
