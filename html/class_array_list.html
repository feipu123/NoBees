Detailed Description

template<class E>
class ArrayList< E >

The ArrayList is just like vector in C++. You should know that "capacity" here doesn't mean how many elements are now in this list, it means the length of the array of your inner implemention For example, even if the capacity is 10, the method "isEmpty()" may still return true.

The iterator iterates in the order of the elements being loaded into this list
