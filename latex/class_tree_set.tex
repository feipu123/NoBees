\subsection{Detailed Description}
\subsubsection*{template$<$class E$>$class TreeSet$<$ E $>$}

A set implemented by balanced tree, the elements being putted must guarantee operator'$<$'

The iterator must iterates in the order defined by the operator'$<$' (from the smallest to the biggest)
